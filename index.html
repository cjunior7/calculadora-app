🧮 Calculadora en C++

Este es un proyecto simple hecho en C++ que permite realizar operaciones matemáticas básicas:

  • ➕ Suma
  • ➖ Resta
  • ✖️ Multiplicación
  • ➗ División

🚀 ¿Cómo usarla?

1. Asegurate de tener instalado g++ en tu sistema

2. Compilá el archivo con:

g++ calculadora.cpp -o calculadora

3. Ejecutá con:

./calculadora
